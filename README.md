# check-links

test-links is a CLI tool written in nodejs for checking broken links in a website

## Installation

Use the package manager [npm](https://www.npmjs.com/package/test-links) to install test-links.

```bash
npm i test-links
```

## Usage

```bash
test-links https://wasif.codes

```
